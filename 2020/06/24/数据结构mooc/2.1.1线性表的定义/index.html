第二章 线性结构

例1：一元多项式的运算（1.1）

一元多项式：f(x)=a0+a1x+…+an-1xn-1+an

主要运算：多项式相加、相减、相乘

多项式的表达：

​ 多项式的关键数据：

​ · 多项式项数n

​ · 各项的系数ai以及指数i

方法1：顺序存储结构直接表示

例如：f(x)=4x5-3x2+1

表示为

下标i 0 1 2 3 4 5 …
a[i] 1 0 -3 0 0 4 …
方法2：顺序存储结构表示非零项

例如：p1(x)=9x12+15x8+3x2

表示为

下标i 0 1 2 …
系数ai 9 15 3 …
指数i 12 8 2 …

相加过程：

从最高项开始，比较两个多项式当前对应项的指数

p1：（9，12），（15，8），（3，2）

p2：（26，19），（-4，8），（-13，6），（82，0）

p3：（26，19），（9，12），（11，8），（-13，6），（3，2），（82，0）

方法三：链表结构存储非零项

链表中每个结点存储多项式中的一个非零项，包括系数和指数两个数据域以及一个指针域。

线性表

基本定义

定义：由同类型数据元素构成有序序列的线性结构

  • 表中元素个数称为线性表的长度
  • 线性表没有元素时，称为空表
  • 表起始位置称为表头，表的结束位置称为表尾

抽象数据类型描述

类型名：线性表（List）

数据对象集：线性表是n（≥0）个元素构成的有序序列（a1,a2,…,an)

操作集：

​ 线性表L∈List,整数i表示位置，元素X∈ElementType(可以为整型，一个实型,一个结构）。

  1. ​ List MakeEmpty(): 初始化一个空线性表L；
  2. ​ ElementType FindKth(int K,List L): 根据位序K，返回相应元素；
  3. ​ int Find(ElementType X,List L): 在线性表L中查找X的第一次出现位置；
  4. ​ void Insert(ElementType X,int i,List L): 在位序i前插入一个新元素X;
  5. ​ void Delete(int i,List L): 删除指定位序i的元素；
  6. ​ int Length(List L): 返回线性表L的长度n。

数组实现

​ 利用数组的连续存储空间顺序存放各元素。

访问下标为i的元素：L.Data[i] 或 PtrL->Data[i]

线性表的长度：L.Last+1 或 PtrL->Last+1 ( 包括0位在内)

1.初始化（建立空的顺序表)
2.查找
3.插入（第i（1≤i≤n+1）个位置上插入为X的新元素）

​ 实际上表示把这个数放在i-1这个位置。

​ 先把（i-1）之后的元素全部移动往后挪（从后往前），再插入新元素。

4.删除（第i（1≤i≤n+1）个位置上的元素）

​ i之后的元素全部往前挪

链表实现

​ 不要求逻辑上相邻的两个元素物理上也相邻：通过“链”建立起数据元素之间的逻辑关系。

​ 插入、删除不需要移动的元素，只需要修改“链”。

1.求表长
2.查找

（1）按序号查找：FindKth

（2）按值查找:Find

3.插入（在第 i-1（1≤i≤n+1）个结点后插入一个值为X的新结点）

​ 先构造一个新结点，指向s；再找到第i-1个结点，用p指向；然后修改指针，插入结点（p之后插入新结点是s）。

4.删除（删除链表的第i（1≤i≤n）个位置上的结点）

​ 先找到链表的第i-1个节点，用p指向；再用指针s指向要被删除的结点（p的下一个结点）；
